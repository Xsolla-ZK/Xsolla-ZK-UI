# Changelog

## v0.0.3 (2023-10-05)

### Added
- Добавлена возможность вставки изображений в компонент `XZKUIRichIcon` с обрезкой по форме SVG.
- Добавлен новый компонент `ConfiguredHeader` для настройки заголовков и подзаголовков в модальных окнах.

### Changed
- Оптимизирован компонент `XZKUIRichIcon` для улучшения производительности и типизации.
- Обновлен Dockerfile для поддержки сборки и развертывания Storybook с использованием `pnpm`.
- Обновлен `deploy.yml` для корректной работы с приватными Docker-образами и аутентификацией через SSH.

### Fixed
- Исправлены ошибки типизации в компонентах `XZKUIRichIcon` и `ComponentButton`.
- Исправлены проблемы с загрузкой приватных Docker-образов через SSH.
